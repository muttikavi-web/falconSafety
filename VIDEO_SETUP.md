## Video Format Requirements
- Format: MP4 (H.264 codec recommended)
- Resolution: 720p minimum (1080p recommended)
- Bitrate: 2-5 Mbps
- Duration: 45-90 seconds for best engagement
- File size: Keep under 50MB per video for fast loading

## Troubleshooting
If videos still don't play:
1. Check file names exactly match: video1.mp4, video2.mp4, etc
2. Ensure files are in: public/videos/ folder
3. Clear browser cache (Ctrl+Shift+Delete)
4. Try on different browser
5. Check browser console for specific error messages
